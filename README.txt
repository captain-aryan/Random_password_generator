RANDOM PASSWORD GENERATOR

Manual:
  -h, --help            show this help message and exit
  -c, --charset CHARSET
  -l, --length LENGTH

USAGE:
-l [LOWERCASE], Enters the lowercase password
-u [UPPERCASE], Enters the uppercase password
-n [NUMBERS], Enters the numbers password
-s [SPECIAL_CHARACTERS], Enters the special characters password

Lenght of the password is must be in numeric format (0-infinite)

Example:
python3 pass_gen.py -c ls -l 10              (prints lowercase and special_chars)
OUTPUT: Random Password is: u+\z,n*?ay
